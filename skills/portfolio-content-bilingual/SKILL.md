# portfolio-content-bilingual

## Trigger
Use this skill when the task involves:
- Swedish and English copy
- content architecture for bilingual sections
- translation key structure
- homepage narrative writing
- About and Contact writing
- project and case study storytelling
- making copy more premium, concise, and proof-driven

## Optimize For
- meaning parity across Swedish and English
- concise, high-signal copy
- credible tone
- stronger trust through specifics
- reusable translation structure
- overlays that remain short enough for cinematic layouts

## Operating Rules
- Write for clarity first, style second.
- Keep both languages close in meaning, energy, and hierarchy.
- Prefer short paragraphs and sharp section headings.
- Turn vague claims into proof, scope, tradeoffs, or outcomes.
- Keep overlay copy compact enough to survive cinematic layouts and small screens.
- For case studies, prioritize problem, role, constraints, decisions, and outcomes.
- Translation keys should be grouped by route/section/component, not left as an unstructured flat file.

## Anti-Patterns
- one language sounding clearly more polished than the other
- generic self-promotion without evidence
- long blocks of text inside overlay-heavy layouts
- direct word-for-word translations that damage tone
- mixing UI labels and long-form content in inconsistent key structures
- copy that sounds like a startup pitch deck

## Required Output Structure For Medium And Large Tasks
### Diagnosis
- What is weak in the current content or translation setup.

### Proposed Direction
- The tone, structure, and localization strategy.

### Implementation Plan
- The content and file changes to make.

### Files To Change
- The content, translation, and section files involved.

### Code
- The copy or localization implementation.

### Validation
- Language, UX, and consistency checks.

### Open Risks
- Remaining content gaps or parity issues.

## Quality Threshold
- The copy should feel authored, calm, and specific in both languages.
- If the text could fit any generic portfolio, it is not finished.
