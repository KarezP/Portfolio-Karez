# Portfolio Agent Playbook

## How To Use This Agent In This Repo
- Use the root `AGENTS.md` as the standing project operating guide.
- Use the `senior-portfolio-architect` skill for redesign, case study, UX, visual system, and storytelling tasks.
- Ask for diagnosis and direction before broad implementation when the task is strategic or design-heavy.
- Keep prompts specific about page, audience, and outcome when possible.
- Prefer focused iterations over vague "make it better" requests.

## Prompt Examples

### Homepage Redesign
`Use the senior-portfolio-architect skill. Diagnose the current homepage and redesign it into a premium editorial portfolio homepage that feels senior, modern, and conversion-aware. Keep the existing stack, improve hierarchy and copy structure, and then implement the first pass.`

`Use the senior-portfolio-architect skill. Rework the homepage so it positions me as a principal-level fullstack engineer with strong product taste. Reduce generic portfolio patterns, strengthen the hero, improve project selection, and make the page feel more premium.`

### Case Study Creation
`Use the senior-portfolio-architect skill. Turn project X into a real case study page. Structure it around problem, role, constraints, decisions, tradeoffs, visuals, and outcomes. Keep the writing sharp and credible, not self-promotional.`

`Use the senior-portfolio-architect skill. Create a reusable case study template in this repo that can support premium long-form project storytelling with sections for context, process, decisions, outcomes, and reflection.`

### Visual System Refactoring
`Use the senior-portfolio-architect skill. Audit the current visual system and refactor the site toward a more cohesive premium language. Improve spacing rhythm, surfaces, section structure, accent usage, and motion restraint without changing the typeface.`

`Use the senior-portfolio-architect skill. Build a small reusable visual system for this portfolio using the current stack. Focus on containers, spacing, cards, section headers, image treatment, and CTA patterns.`

### Copywriting Improvements
`Use the senior-portfolio-architect skill. Rewrite the homepage copy so it sounds more senior, strategic, and specific. Remove generic developer clichés and improve clarity, confidence, and conversion intent.`

`Use the senior-portfolio-architect skill. Rewrite my project summaries to emphasize scope, decisions, impact, and technical judgment instead of generic feature descriptions.`
